## 🔗 요구사항 ID 체계

### 명명 규칙: `REQ_[도메인]_[그룹번호]`

- **USR**: User (사용자 관리)
- **BMK**: Bookmark (북마크 관리)
- **RCP**: Recipe (레시피 관리)
- **NTR**: Nutrition (영양정보 관리)
- **WEB**: Web Interface (웹 인터페이스)
- **SYS**: System (시스템 아키텍처)
- **SEC**: Security (보안)
- **DOC**: Documentation (문서화)
- **MON**: Monitoring (모니터링)

### 상세 ID 규칙: `REQ_[도메인]_[그룹번호]_[순번]`
